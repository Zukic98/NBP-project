# Advanced Databases - JDBC Project

This repository contains a Java-based application developed for the **Advanced Databases** course. The project focuses on direct database interaction using the **JDBC API**.

## Tech Stack
* **Language:** Java (JDK 11+)
* **API:** JDBC (Java Database Connectivity)
* **Database:** [PostgreSQL]

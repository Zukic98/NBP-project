# Frontend Setup

Follow these steps to run the application locally.

## Prerequisites
- Node.js installed

## Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Zukic98/NBP-project
   cd frontend

2. **Install dependencies:**
   ```bash
    npm install

3. **Environment Variables:**
   ```bash
    cp .env.example .env

4. **Start the development server:**
   ```bash
    npm run dev
